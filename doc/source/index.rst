==============================
lb_toolkits使用说明
==============================

.. note::

    由于团队成员减少，功能开发、测试、发布受到影响，
    将争取每个月发布一版，增加遥感数据处理、气象计算公式、
    栅格矢量基础操作等功能。

    未来计划：

    - 风云卫星产品数据的处理、图像绘制等功能
    - 气象行业常用的计算公式
    - 常用的散点图、直方图、时序图绘制等

.. warning::

    当前 ``lb_toolkits`` 库还不是一个稳定的版本，正在对接口API
    进行测试补充，争取把接口满足大家需求。
